# Dunajec 2026

Staticky web pro facebookovou udalost tridni vody na Dunajci.

## Soubory

- `index.html` - hlavni stranka
- `styles.css` - vzhled
- `script.js` - prepinani dni, variant a kopirovani textu rezervace
- `itinerar-dunajec-2026-overeno.md` - overeny obsahovy podklad a zdroje

## Publikace

Web je pripraveny pro GitHub Pages nebo libovolny staticky hosting. Vstupni soubor je `index.html`.
